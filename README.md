# To-Do List App
A Simple To-Do List app built with Redux

### Clone
Clone the Project

```sh
$ git clone https://github.com/axissixa57/rss-todo-app-redux.git
$ cd rss-todo-app-redux
```

### Installation

Install the dependencies

```sh
$ npm install
```

### Run

This will run your dev server at http://localhost:3000

```sh
$ npm start
```
